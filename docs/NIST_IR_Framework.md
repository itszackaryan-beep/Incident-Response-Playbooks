
# NIST Incident Response Framework

## Overview

The NIST Incident Response Framework provides a structured approach for detecting, responding to, and recovering from cybersecurity incidents. This project follows the guidance of NIST SP 800-61 Rev. 2 to create incident response playbooks for ransomware, brute force, and data exfiltration scenarios.

---

## 1. Preparation

Prepare the organization by implementing security controls, defining response procedures, and ensuring that security teams have the necessary tools and training.

---

## 2. Detection and Analysis

Monitor security events, investigate alerts, analyze logs, and confirm whether a security incident has occurred.

---

## 3. Containment, Eradication, and Recovery

Contain the incident to prevent further damage, remove the threat, recover affected systems, and restore normal operations.

---

## 4. Post-Incident Activity

Review the incident, document lessons learned, and improve security controls and response procedures to reduce future risks.

---
